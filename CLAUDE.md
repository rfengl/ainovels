# CLAUDE.md — 小说创作项目须知

> 新 session 开工前：**先读本文件 + `CORE_WORLD.md`（设定真理来源）+ `OUTLINE.md`（全书纲要）**，再动笔。

## 这是什么
一部中文网文长篇。调性：轻松搞笑为主、爽文为辅。

## 目录与命名
- 章节文件：`HH/TTT/XXXX.md`，**两级目录，每个内层文件夹 10 章**（首个文件夹因含序章 0000，多收一章共 11 章）：
  - 内层 `TTT` = `floor((章号-1)/10)` 补 3 位（序章 0000 归 `000`）；外层 `HH` = `floor(TTT/10)` 补 2 位。
  - 例：`00/000/` = 0000–0010（11章），`00/001/` = 0011–0020，…，`00/009/` = 0091–0100，`01/010/` = 0101–0110，`02/029/` = 0291–0300。
- `0000.md` 是序章。`CORE_WORLD.md` = 设定；`OUTLINE.md` = 大纲。

## 写作铁律（每章必守；细则见 CORE_WORLD.md §12）
1. **每章正文 ≥ 2000 字**（中文字符含标点；用 `wc -m` 计，与 PowerShell `.Length` 一致）。
   - **唯一例外：序章 0000** —— 只求"前两段强勾子 + 最佳阅读体验"，**不凑字数**，短一些无妨。
2. **慎用套路化身体反应词**：`一僵/一愣/愣住/怔住/攥紧/握紧拳头/指甲嵌入(掐进)掌心/瞳孔一缩/心头一(震跳沉)/鼻子一酸/眼眶一热/喉咙发紧/喉结动了动/后背(脊背)发凉/深吸一口气/嘴角勾起一抹弧度/鬼使神差/头皮发麻/苛刻`。每章最多 1 次，优先用**具体动作、对话、内心吐槽、留白**代替。
3. **段落要短**：一段 1–3 句；对话各自独立成段；长段（>4句或~120字）拆开；多留白。扩写靠"加场景"，不靠把句子堆进一段。
4. **尽量不使用破折号（——）**，适当使用逗号、句号或省略号。
5. **产生新设定就回填 `CORE_WORLD.md`**，保持唯一真理来源。

- 提交前用脚本守门：字数 >= 2000（序章除外）不要提交，先补内容。
