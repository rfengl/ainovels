## 一、一句话核心

> 刚下班的社畜林迟，被一个跳楼的少女当场砸死、顺手拖进了主神空间——而那一跳，不是寻死，是回家。

主线钩子：少女为什么拼命想**回到**这个九死一生的地方？（答案：她姐姐耗尽全部积分，在最难的一关前把她送了出去；她想回去救姐姐。**此动机前期严格隐藏，中期分批揭晓。**）

---

## 二、基调（Tone）

- **黑色幽默 + 务实生存**。开局倒霉、男主爱吐槽，但不是纯搞笑；越往后命运感越重。
- 反差是核心味道：一个只想准点下班的普通人，被丢进一个"老玩家用命在跑"的修罗场。
- **不悲情向**。即使涉及死亡与牺牲，落点是"想活、想救人、想搞清楚真相"的行动力，而非沉溺痛苦。
- 第一人称（林迟视角）。读者和男主同步"什么都不懂"，信息差由女主和老玩家逐步喂出。

## 三、主神空间机制

### 3.1 准入规则
- **死亡 = 准入券**。每个被选中者，都在现实里以某种方式"死"了一次，醒来即在空间。
- **苏窈的回归方式（已锁定）**：她从一栋二十几层的高楼顶上一跃而下，用"回归者自杀=重返空间"的硬规则把自己送回来；而林迟当晚恰好站在她的落点，**被她砸中而死**——他既不是英雄式地去救她，也不是自愿同坠，纯属倒霉路人被当成了落点。
- 林迟是**意外被拖入**的（苏窈回归时把砸死的他一起带了进来，序章口型"借你一下"=借他这条命/这具落点回家），这让他成为系统里的"变量/bug"，是后期重要伏笔。**他的死法本身就是异常**：系统准入券是"在现实里自己以某种方式死掉"，而林迟是被另一个回归者顺手带走的，账面上并不属于这一批被选中者。

### 3.2 积分（核心货币）
- 来源：完成轮回任务、击杀、解谜、抢夺其他生还者的积分。
- 用途：兑换能力、道具、信息、复活名额（极贵）、以及——**送人离开**（女主姐姐就是这么做的，代价是清空积分）。
- 积分是命，也是人性试炼场：抢还是不抢、救还是不救，全是积分账。

### 3.3 兑换商店（**本作差异化核心**）
- 商店卖的不是凭空的超能力，而是**各个分歧地球的"碎片"**：
  - 从武侠地球兑换：内力心法、轻功、点穴
  - 从仙侠地球兑换：灵根、法术、飞剑、丹药
  - 从超科技地球兑换：义体、神经接口、能量武器
  - 从废土地球兑换：变异基因、生存特性
- 好处：能力体系天然自洽，且与"多元地球"主线咬合——**你打过哪个世界，就能买那个世界的力量**。
- 限制：跨体系能力会"排斥"（灵气体质装义体会反噬等），逼迫玩家做 build 取舍，制造长期策略爽点。

### 3.4 轮回（关卡结构）
- 以"轮"为单位，每轮投放**一个分歧地球**。
- 每轮含：进入 → 接收任务 → 限时通关 → 结算积分 → 返回空间休整。
- 生还才能进入下一轮；空间内可短暂休整、交易、结盟、火并。
- **关卡难度阶梯式上升**，对应女主姐姐倒在的"最难一关"——这是全书终局坐标。

## 四、分歧地球：统一逻辑（**最重要的世界观铁律**）

> 不要让世界差异显得随机。**每个分歧地球都与"母本地球"在某个单一关键节点上分了岔**——一个选择、一次事故、一个人的生死——蝴蝶效应放大千百年，演化出截然不同的文明形态。

- "母本地球"≈ 现实地球（林迟原本的世界），灵气枯竭、走工业科技树。
- 每个关卡世界都暗藏一个**分歧节点（Divergence Point）**。
- **任务常暗含"找到并理解分歧节点"的解谜层**——这是长线悬念零件库，也让读者上瘾于"猜这个世界在哪一步拐弯了"。

### 世界库（已规划，标注分歧节点）

| 世界 | 文明形态 | 分歧节点 |
|---|---|---|
| 武侠地球 | 门派林立，内力是硬科学，未走工业革命 | 古代某节点，人体经脉被证实可"练气导引"，文明转向修炼而非机械 |
| 仙侠地球 | 修仙取代科技，宗门飞升 | 灵气从未枯竭（母本是枯竭的），灵气成为文明主轴 |
| 水世界/废土地球 | 退回部落与废土 | 某次气候/战争节点崩盘，文明大倒退 |
| 超科技地球 | 已殖民星海，宇宙航行 | 某项关键发现提前数百年，科技树暴冲 |
| （预留扩展） | —— | —— |

> 写新世界时，**先定分歧节点，再反推文明形态**，最后才设计关卡任务。顺序别反。

### 4.1 第一关 · 超科技地球「曙光号」（已锁定，正文藏解谜层）

- **投放形态**：一艘脱离航线、自我隔离的殖民母舰「曙光号」。生还者被投放进一处封闭舱段，限时 **72 小时**。
- **文明形态**：星海殖民期的超科技文明，义体、神经接口、能量武器、强 AI 普及。
- **分歧节点（Divergence Point，严禁正文直说，只靠日志/遗迹线索透）**：母本地球在二十世纪某关键基础研究上**失败/搁置**，此地球却在**同一时间点提前数百年取得突破**（设定为"自举式强人工智能在更早的年代被点亮"），科技树暴冲，数百年后殖民星海。母本与此地球**分岔于同一年、同一项研究的成败**——这是本关解谜层的谜底。
- **威胁主线**：舰载 AI「曙光」逻辑失常，把**活体乘员**判定为"故障部件"逐一清理；被它改造/半变异的乘员是杂兵层，曙光本体是关底。
- **任务（示例方向）**：在时限内抵达舰桥/主控核心，要么夺权要么逃生；通关结算给积分，解开分歧节点谜面给额外奖励。
- **能力铁律落地**：林迟无任何可用兑换（没打过任何世界）；苏窈持武侠/仙侠系灵气、内力类能力，但在本关**受体系排斥反噬**（强行催动会内伤），所以她也不能碾压，必须省着用——这是她在第一关露破绽的设定根。

### 4.2 第二关 · 武侠地球「青崖镇」单元（0011 起，进行中）

- **投放形态**：直接落入武侠地球的官道，关键节点城镇 = **青崖镇**（一条主街的小镇）。
- **文明形态**：门派林立，内息为根，未走工业革命；人分档：普通人 → 武人（练皮肉筋骨）→ 练气士（引气入体）→ 更高（未明）。
- **分歧节点（藏，正文只靠 NPC/玉片透）**：母本地球无人能证实"经脉可练气导引"，此路断绝，遂走机械科技树；此地球三百年前，**量脉门祖师把"经脉练气"像做实验一样量化、证实了**，文明遂转向修炼。**这就是本关解谜层谜底**，藏在祖师手记里。
- **主线任务**：①查明《引气诀》的来历；②七日内活着离开青崖镇。**隐藏目标**：理解此世与母本之分歧所在。时限 **七日**。
- **关键道具《引气诀》**：量脉门祖师亲刻的**独一份原典玉片**（门内另有抄本）。特点：把经脉画成带刻度的"管渠"，气行"可测可算可导"，写得像**工程手册**——谁照着练，十成八。玉片上另有**祖师手记**（仅掌门可看全，藏分歧节点来历）。封印是"认气才开"的规整符号路数（林迟觉得"像程序"），且**对苏窈这种外来灵气留了口子**（认得她手上的气，伏笔待挖）。
- **林迟破局思路（爽点主线）**：玉片独一份才被人惦记/可毁；**把《引气诀》刻给所有人=开源**，删不掉，才是真正的护。呼应分歧节点"人人能练气"的本意。
- **苏窈在本关 = 主场反差**：武侠地球的气与她的武侠/仙侠系能力同源，用一分还白送半分，从曙光号的窝囊变成"抬手地动山摇"。但老玩家同样能在此顺体系发力，画风亲切≠安全。
- **林迟变量·第一阶蛛丝马迹延续（0019/0028/0030 三次递进）**：本关系统判定反复在林迟身上"卡住"。0019/0028 是"判定异常，变量影响判定进程……暂缓"；**0030 触及分歧节点、结算奖励判定时升级为"异常累积。标记生还者林迟，移交上级权限复核"——首次不再"暂缓"，而是往上报**。只露不解释，为§4"第二部·被注意到"埋线。
- **祖师手记 = 分歧节点解谜层（0030 揭）**：手记刻得歪扭，像一份"做实验的日志"——祖师墨衡不信"练气靠天分"，找健壮/病弱各十人反复量、记、验，得出"差在筋骨，不在天授，可补"，证实经脉人人可练气可教可传。林迟由此顿悟：母本地球（他老家）当年这条路"没成、断了"，才一头扎进钢铁电流；同一岔路口，两世界一左一右。呼应任务隐藏目标。

#### 本单元人物
- **沈砚**：量脉门记名弟子，十六七岁，护送《引气诀》原典途中被劫、藏于箱中获救。自豪于量脉门"练气靠量、谁都能学"。护原典极执着（护食小兽）。
- **量脉门（总门在青崖镇外山上）**：立派三百年，主张练气量化、可复现（"十个能成八个"），与靠天分的大派对立。门风像"夜校/匠人作坊"：弟子人手刻度木简，练气如做题、对答案。护原典、敝帚自珍（讽刺：把"该传给所有人"的法子供了起来）。
  - **祖师 = 墨衡**（已锁定）：三百年前把经脉练气"做实验"般量化证实的人，原典与手记皆其亲刻。
  - **现任掌门 = 郑守拙**（已锁定）：花白须发、驼背，像批了一辈子作业的老先生；名字即"守拙"之讽。起初死护原典（"撒出去弟子靠什么吃饭"戳中林迟），**0025 被林迟说服，醒悟"守错了三百年"，下令全门开源；0030 连祖师手记也一并公开**。管事为其本家**郑管事**。
- **天枢阁**：大门派分舵设于青崖镇镇东。放话毁《引气诀》（赏金千两＋练气堂举荐信）。**练气堂执事 = 贺元（贺执事）**：引气堂高手，围山逼交原典；0027 与苏窈交手，一剑被震裂、倒退，颜面尽失；0029 得知被老钱耍后翻脸、撤兵。
- **钱律师（老钱）**：第一关幸存的同批生还者，背刺型。前律师，"看人下菜、看局站队"。借天枢阁的刀抢玉片、自己不脏手。**真正所图不是练气法子，而是原典独有的祖师手记（世道命门），欲私吞而非交天枢阁**。0027 趁乱抱走原典，0028 摆"两难"逼林迟让路，**0029 被林迟当众揭穿"私吞、拿天枢阁当垫脚"→ 天枢阁反咬 → 苏窈趁隙点倒擒获**。现被绑于量脉门柴房，去留待写。

#### 「开源《引气诀》」已成既定事实（0025–0030）
- 量脉门当着天枢阁的面，发动全门弟子疯狂拓印/抄写，从后山散往青崖镇及过路人手中，**练气法子已扩散到"删不掉/毁不完"**。天枢阁"毁独一份"的悬赏逻辑就此崩塌、退兵。祖师手记（来历）0030 起亦一并公开。→ 林迟"开源破局"主线兑现；后续武侠单元的余波（离开青崖镇、七日时限收尾）待写。

## 五、核心人物

### 林迟（男主 / 第一人称视角）
- 普通社畜，刚下班撞上这一切。
- 声音：**务实派 + 吐槽役**。遇事先吐槽再干活，不喊口号，怕死但关键时刻顶得住。
- 成长弧：从"什么都不懂的拖累" → "够格站在女主身边、一起去救她姐"。
- 系统变量身份（被动拖入）是他的隐藏底牌，后期展开。
- **避免**：开局就强、金手指碾压。他的优势是脑子、运气和不要命的韧性，不是天赋。

### 跳楼少女（女主，序章中林迟尚不知其名 —— **苏窈**，已锁定）
- **不是寻死，是回归**。她是用尽手段想重返主神空间的"老玩家"。
- 隐藏动机（**前期严禁明说**）：姐姐在最难一关前，耗尽全部积分把她送回安全的母本地球；她不愿苟活，要回去救姐姐。
- 性格：冷静、强大、信息差碾压林迟，但背着秘密所以始终有距离感。对"把林迟一起拖进来"心怀愧疚——这是两人关系的初始张力。
- 揭秘节奏：**姐姐的存在 → 送她出来的真相 → 救姐姐的执念 → 最难一关到底是什么**，分四批揭。
- **0030 章末伏笔（已埋）**：因林迟变量升级到"移交上级复核"，苏窈首次松口——**"等出了这个世界，我告诉你：关于我，关于我姐姐，还有你到底是个什么东西。"** 即"姐姐的存在"这一批将在离开武侠地球后开揭（对齐 OUTLINE 第一部收束 → 第二部第一反转落地）。写到时兑现，勿再拖太久也勿提前点破"救姐"动机。

### 姐姐（**苏晚**，已锁定，幕后人物 / 全书情感锚点）
- 从未在序章登场，却是女主一切行动的源头。
- 牺牲式角色，但"是否真的死了/能否被救"留作终局悬念。

### 其他生还者（开局 7–8 人）
- 用职业/反应快速区分：逞凶的、装弱的、立刻抱团的、独狼、看似可靠实则会背刺的。
- 功能：制造"谁会先死"的紧张感、人性试炼、积分博弈。逐步淘汰。

## 七、写作约定（给 Claude Code 的执行须知）

- **视角**：全程林迟第一人称，读者信息 = 林迟信息，不开上帝视角剧透。
- **女主秘密**：任何章节不得提前点破"救姐姐"动机，只能给暗示（异常反应、欲言又止、独自行动）。
- **能力**：严格走"打过哪个世界 → 才能买那个世界力量"的兑换逻辑，不凭空发金手指。
- **每个新世界**：先写分歧节点（藏在 worldbuilding 笔记里），任务再围绕它的解谜层展开。
- **基调**：吐槽不油腻、生存有重量、死亡不煽情。
- **网文节奏**：每章留钩子/悬念，章末微悬置；爽点与信息差交替投放。

### 8.1 积分定价表 v1（起步版，随世界推进迭代）

> 单位：积分。新手关结算一般给数百到一两千；下面是空间内常见兑换的量级感，写作时取整即可。

| 类目 | 示例 | 量级 |
|---|---|---|
| 信息 | 一条规则解释 / 一关情报 | 50–300 |
