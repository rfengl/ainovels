## 0. 一句话设定(Logline)

一个天生听得懂灵宠说话、却只想躺平养可爱灵宠过日子的御灵师,在毫不知情的情况下,一路捡回的全是退化成团子的上古神兽幼崽。

## 1. 书名

- **主名:**《灵宠太爱操心了》
- **定调结论:** 长线生命力在「治愈 + 喜剧」。

---

## 2. 核心设计原则(写作时务必守住)

**「只收可爱」不是女主给自己上的难度,而是一个被全世界误判的隐藏外挂。**

1. **可爱 = 幼态神兽。** 神兽、凶兽在「幼态 / 返祖前」一律长得人畜无害、软乎乎。主流御灵师嫌弃幼态萌宠=废柴,女主天生对可爱没抵抗力,于是无意间一路在集邮未来神兽。
2. **反差是取之不尽的爽点。** "我只是图个可爱" ↔ "我家随便一只都是上古大佬",打脸可以反复制造,且不靠数值碾压。
3. **爽点引擎是『养出来的』而非『刷出来的』。** 力量来自羁绊(见 §4 契深机制),抗长线疲劳。这是本书区别于普通爽文的命根子,不要为了短期爽感破坏它。
4. **情感优先于战力。** 女主能听见灵宠惨叫 → "收集"天然带正义与温情底色,不是单纯凑数。

## 4. 力量体系

### 4.1 契深机制(全书力量命门)

灵宠战力 = 灵宠资质 × **契深(羁绊深度)** × 主人御灵师阶位。其中契深是关键变量:

| 契约层级 | 缔结方式 | 战力上限 | 风险 |
|---|---|---|---|
| 寄契 | 武力压服 / 强契 | 封顶 | 灵宠心怀怨恨,易反噬、暴走、背契 |
| 共契 | 互利合作 | 中等 | 利尽则散 |
| 心契 | 真心相待 | **无上限**,解锁隐藏进化 | 无 |
| 神契 | 生死与共 | **触发血脉返祖 / 觉醒** | 反噬生主(双向) |

- **主流御灵师**为求快普遍用 **强契** 抢强宠 → 又强又恨主人,封顶且随时背刺。
- **女主**靠听懂、靠真心,轻松达到别人一辈子摸不到的 **心契 / 神契** → "废柴萌宠"能一路觉醒、突破天花板。**这就是她吊打天才的底层逻辑。**

### 4.2 灵宠阶位

一阶 凡灵 → 二阶 灵兽 → 三阶 妖灵 → 四阶 灵将 → 五阶 灵王 → 六阶 灵皇 → 七阶 灵尊 → **神兽(传说阶,不入常规阶位)**

> 关键:**幼态神兽常被仪器/主流认知误判为「一阶凡灵」**,这正是女主能"捡漏"的机制来源。

### 4.3 御灵师阶位

学徒 → 御灵师 → 灵御 → 灵宗师 → 灵尊 → 灵主

> 御灵师晋阶受最高契深 + 契宠阶位牵制 → 女主靠"少而精 + 高契深"弯道超车。

## 5. 女主能力:言灵

- 普通御灵师只能感知模糊的 **兽意**(大概的情绪)。
- 女主能听见任何灵宠 **完整的话**:契约的、野生的、昏迷的、甚至别人的契宠。
- 用途:谈判、招安、套情报、拆穿强契宠的怨恨、关键时刻策反敌方契宠。
- **稀有度:** 每一代只出一个 **言灵者**。来源是贯穿全书的大谜团,挂钩远古 **灵祖**。

## 6. 女主:沈宁

- **本名:** 沈宁(含"图个安宁、好好过日子"之意)。
- **人设:** 低欲望、犯懒、嘴上吐槽心里软。在人人卷战力榜的世界里,唯一一个"修仙是为了躺平养宠"的异类——反差本身即新鲜的喜剧声音。
- **底色:** 表面咸鱼,遇灵宠受虐会瞬间硬气,因为她是唯一听得见它们惨叫的人。
- **性格配比(已定):** 以**咸鱼为主**——低欲望、犯懒、嘴上吐槽,人生理想是躺平养可爱;唯一的硬气开关是**灵宠受虐**(只有她听得见惨叫)。所谓"腹黑"**不是主动算计,而是"懒得解释"**:她随口一句、随手一养,旁人自行脑补成深不可测的神操作 → 打脸由误会驱动。爽而不油,不破治愈基调。

## 7. 灵宠图鉴(签名级五只,可无限扩)

> 写作要点:每只灵宠先用**具体的身体细节和说话口癖**立住,再揭隐藏真身。少用旁白概括性格。

### 糯糯(本命第一宠)
- **外形:** 巴掌大的白胖团子,像长了四条小短腿和一对豆豆眼的汤圆,走路一颠一颠。
- **性格:** 胆小、贪吃、话痨、戏精。口癖:"糯糯不行了……糯糯要饿死了……"
- **真身:** 幼态饕餮。贪吃实为 **吞噬血脉**,觉醒后能吞万物、吞攻击。
- **定位:** "怂萌团子 ↔ 混沌吞兽"的招牌反差,"可爱=神兽"引擎的样板宠。

### 玄夜(别扭担当)
- **外形:** 巴掌大的黑奶猫,琥珀色竖瞳,尾尖一撮白。
- **性格:** 傲娇暴躁,坚称"本座只是暂居于此";嫌弃抱抱,被冷落又生闷气;对别的宠醋意滔天;偷睡她枕头死不承认。**本性极护主**。
- **真身:** 被封印的堕落妖王。别扭源于它记得自己曾经多威风,如今却是个毛球。
- **定位:** 用户点名要的"别扭但本性不坏",由它扛主力。

### 咕噜(老干部反差)
- **外形:** 青色圆滚小团子(小石灵 / 小奶龟),慢吞吞。
- **性格:** 说话文绉绉一股老干部味,自称"老夫";被 coo、被戴小帽子时尊严扫地却无可奈何。
- **真身:** 重伤退化的上古古神。
- **定位:** 讲世界观 lore + 一本正经的反差喜剧。与玄夜区分:玄夜是暴躁少年感,咕噜是慈祥老者。

### 暄暄(治愈担当)
- **外形:** 金毛绒球小幼崽,口水多(常挂一长串亮晶晶口水)。
- **性格:** 金毛犬能量,不太聪明,把沈宁当神无脑崇拜;打架勇猛但常帮倒忙。力气大得离谱却完全不会控制(一脚蹬塌破院栅栏),闯了祸就"哇"地哭、生怕被丢下。被原主人当废物嫌弃丢弃,却毫不难过,只认准"能让大家都不害怕的人"。
- **真身:** 圣兽幼崽,血脉"越护主越强"。
- **登场(第43章,第二卷开篇):** 被原主人当废物丢弃,守在沈宁破院门口数日,凭"味儿"认定沈宁,死赖着求收留。沈宁收为家中**排行老五**(老大糯糯、老二玄夜、老三红豆、老四寒霜狼、老五暄暄);因其整天乐呵呵暖烘烘"像晒得人发暖的太阳",得名**暄暄**(呼应糯糯=汤圆、红豆=豆,软萌系命名)。真身未揭。

### 乐乐 & 愁愁(相声担当)
- **外形:** 一条巴掌长、灰扑扑的双头小蛇,共用一个身体(一个胃),拆不开。左头乐乐昂着、眼睛亮;右头愁愁耷拉着、眉眼垮。
- **性格:** 左头乐乐凡事乐观("会有好日子的!"),右头愁愁凡事悲观("我们要死啦…全完啦…"),永远在吵架。被世人当"灾星双头蛇"(见双头蛇者不祥)。
- **真身:** 阴阳 / 两仪之力,合体时为隐藏神兽。咕噜见之喃喃"一体两魂,一阴一阳,一动一静"(第47章低声伏笔,真身未揭)。
- **登场(第45–47章,第二卷):** 被笑面货郎(黑市探子)当"套话由头"摆在外门摊上,沈宁循惨叫(只有她听得见两头从早吵到晚)一眼相中,以古长老所赏三颗灵石之一买下(货郎反常低价,巴不得她带走以顺藤摸瓜)。沈宁懒得深究、收为家中**老六**(共一身算一个老六,又俩脑袋各算半号→乐乐老六、愁愁也老六的排号喜剧)。愁愁"丧了一辈子,头一回有人说'谁说的'",入家当晚头回睡在"不会被丢出去的地方",暖得丧气使不上劲。

### 锅巴(自恋担当)
- **外形:** 巴掌大的小鸟,买来三天就掉得没几根毛,粉皮露在外像没下锅的雏鸡;仅剩几撮红一焦一、烤糊般的歪毛,头顶左二右一三根不对称翎子。
- **性格:** 自我感觉极佳、自封"绝世风姿/倾世容颜",对镜孤芳自赏,张口"本凤"。**自恋是壳,内里是被当摆设买来、又因长得丑被嫌弃的孤单与自卑**——越自夸越因为有人骂它丑。认死理:只认听得懂它的人。一肚子戏、最难伺候、最挑食的金贵嘴(要吃热乎的、要单独开小灶)。
- **真身:** 幼态凤凰 / 朱雀血脉(火属性·涅槃)。咕噜见之喃喃"南方丙丁,羽翎未丰,离火未燃,是涅槃之兆,还是……"(第53章低声伏笔,真身未揭)。
- **登场(第49–53章,第二卷):** 内门富家弟子**卫珩**花两百颗灵石淘来当摆设(图血脉名头响、挂战兽榜有面子),买回三天掉毛被嫌"赔钱货",遂绝食硬撑体面(嫌粗食硌牙、骂满院俗人),三位驭兽堂师傅查不出病。沈宁为赚灵石买米接活(治好赏二十颗灵石),一听即知它没病、纯是只自我感觉良好的鸟在孤单赌气;蹲下专戳它最在意处('最好看的凤凰是饿出来的?吃饱长好毛才有脸说绝世风姿,谁再说你丑你啄他')哄它开吃。误会式打脸:卫珩惊为'跟凤凰讲道理'的神仙,付二十颗灵石。次日沈宁一走它又绝食(全世界只沈宁听得懂它),卫珩抱着两百颗灵石的无底洞如释重负把鸟'送给高人',沈宁叹气接纳为家中**老七**。因一身焦黄没长匀的毛,得名**锅巴**(自封'朱炎/凤鸣'誓死不从,一句'免今晚小灶'即认命,反差萌;呼应糯糯=汤圆、红豆=豆软萌系命名)。沈宁赚灵石本为多买米,米没买倒先添张最挑食的嘴'我怎么觉得我亏了'。

### 酥酥(社恐担当)
- **外形:** 巴掌大的小狐狸,一身雪白软绒(白得发亮,像撒了糖霜的点心),耳朵总往下耷拉,又大又蓬的尾巴爱裹住自己、紧张就埋脸进去,湿漉漉的圆眼睛。
- **性格:** 社恐到极点,口癖"对不起…对不起…""求求你们别看我""要是我没出现/没生下来就好了";一被注视/被吓就慌、把自己缩成一团甚至变透明想从世上抹掉自己;笃信"我活着就是个错、会给人添麻烦"。胆小话多,跟同样胆小的糯糯意外投缘。
- **真身:** 幼态天狐 / 灵狐(惑心之力)。幻术随情绪走:**害怕时失控放出吓人的惨白狐影虚张声势,放松时不自觉绽出莹白轻盈的花影**。咕噜见之喃喃"惑心:障眼是表,惑心是里,能乱人眼,更能安人心,好一脉天狐血裔怎落这般田地"(第57章伏笔,真身未揭)。
- **登场(第54–58章,第二卷):** 原是稀罕灵狐血脉、小时被主人宝贝(指望长大放幻术斗法迷惑对手),却因胆太小一上场就吓得幻术失控、把自己人全用幻象困住害主人当众惨败丢脸,被弃骂"晦气灾星、天生克主",自此躲到没人处不再连累谁。沈宁出门换米抄近道,在外门西头废弃柴房撞见缩成一团、抖如筛糠、满口道歉的它;几个外门弟子追来要"剥皮卖钱报当众丢脸之仇",小狐惊恐失控放出满屋幽红眼狐影吓退众人(沈宁一眼看穿是假的、虚张声势,神色如常被脑补成高深莫测——误会式打脸),半哄半拎带回。沈宁点破"幻术问题不在它身上:你怕它就吓人,你不怕它就好看",哄它吃下久违的一整顿饱饭。因一身白软像撒糖霜的点心,得名**酥酥**(久未听过这么软暖的名字,笑着哭),收为家中**老八**。庇护:被当"灾星"丢弃、笃信自己存在即错的孤单小东西获救,头一回不再说"对不起"。

### 雪糕(高冷担当)
- **外形:** 巴掌大的小白鹿,一身雪白软绒(白得发亮、无一丝杂色,凉丝丝像一整块没化的雪糕),细细四条腿,额心未生角、蹄下不踏火。
- **性格:** 高冷骄傲、死要面子,口头"本座""血脉尊贵""与本座无关""你别误会";生而被供奉膜拜,把"尊贵"当壳裹住自己,越狼狈越昂着脖子硬撑(被夹住腿/摔进篓里也要端着"本座一时不慎")。**壳是骄傲,内里是只受了伤、强撑体面、从不肯求人也不肯欠人的孤独崽**——所有靠近它的人眼里都写着"值钱",于是它学会谁也不靠近;沈宁"顺手救、多双筷子、半句不图回报"恰好拆了它的壳(没有债要还,便不必昂着)。仁兽本性:对更弱小的(酥酥/糯糯)刺不出口。
- **真身:** 幼态麒麟 / 瑞兽(仁兽之首,瑞气之尊)。咕噜见之喃喃"一身雪白无杂毛,额心未生角,蹄下不踏火,仁兽之相、瑞气之尊、麒麟之裔,怎么也落到了猎户的铁夹里"(第61章伏笔,真身未揭)。
- **登场(第59–63章,第二卷):** 沈宁为省灵石、独自去外门后山背阴坡割野生灵米(那片地因黑市野猎户下铁夹捕稀罕幼兽、弟子绕行),撞见被生锈捕兽夹困住、腿快废了仍昂头硬撑"区区凡铁也敢困本座"的小白鹿;沈宁掰夹救出、它死要面子三次摔倒仍"歇够自会离去",终别扭"勉为其难随你去歇两日"栽进灵米篓(四脚朝天露白肚)被背回家。回破院一窝活宝迎新口(糯糯热情、暄暄"像一大块好吃的"凑脸、锅巴争"谁更高贵"被怼"越没有越爱挂嘴上"、酥酥怯生生搭话它刺不出口、咕噜埋麒麟伏笔);下夹猎户追上门讨"祥瑞鹿"放威胁,沈宁懒得废话一句"它要走我不拦,你问它",雪糕寒意竖毛、一窝门神(红豆火纹/寒霜狼结霜/玄夜千年杀意一闪)齐刷刷抬头,猎户连滚带爬吓跑(沈宁其实什么没做——又一桩误会式打脸)。夜里点破降服:沈宁不图它祥瑞、不图它血脉稀罕、半句不要回报,雪糕头回明白"原来可以不必昂着",红了眼仍嘴硬"再多留几日是这院子还算清净"。因又白又凉像没化的雪糕,得名**雪糕**(自封"白鹿君/瑞云尊者"誓死不从,一句"免今晚灵米羹"即认命,反差萌;呼应糯糯=汤圆、酥酥=点心软萌系命名),收为家中**老九**。庇护:被当"值钱货"猎捕、学会谁也不靠近的孤傲瑞兽获救,头一回睡得踏实。

### 果冻(爱哭担当)
- **外形:** 巴掌大的小鲛人崽,上半身像个雪白圆乎乎的小娃娃(湿漉漉的大眼睛),下半身一条半透明、泛着淡淡水光的小鱼尾;软软凉凉,泡在水里晃晃悠悠像一块果冻。额角有两点几乎看不出来的小凸起。离了水也能活,只是待在水里觉得安心。
- **性格:** 玻璃心、爱哭、一碰就化,凡感动/委屈/害怕皆掉泪;泪落水中"叮"地凝成珍珠。口癖"我不哭…我这就不哭了""对不起我又掉珠子了""我配吗"。**爱哭是表,内里是被当成『聚宝盆』、只因眼泪值钱才被人围着的孤独**——它一度以为活着的全部意义就是哭、就是给人掉珠子,甚至盼自己『不值钱』『不会哭珠』好确认有没有人单纯因为它本身而要它。沈宁『不要它的珠子、把珠随手扔进装石子的破石臼、说哭也好不哭也好都是我家的人』恰好拆了它的壳(眼泪第一次只是眼泪,不是钱)。
- **真身:** 幼态真龙 / 行云布雨之裔(水·司云雨)。咕噜见之喃喃"鲛人泣珠……可这珠里头缠着的分明是行云布雨的气数,水族之中能凝雨为珠的怎么也落到那等腌臜水缸里"(第67章伏笔,真身未揭)。
- **登场(第64–68章,第二卷):** 九口破院食量危机到顶,沈宁揣锅巴灵石折的碎银去东市买米,巷口撞见黑市贩子用半人高水缸养着、以银针逼哭采珠的小鲛人崽(围观哄抢叫到五十灵石);沈宁循只有言灵者听得见的哭声相中,蹲下一句"别哭了,我不要你的珠子",鲛人头回听见有人不为珠子,死咬唇硬不掉泪,缸主连戳几针逼不出珠、聚宝盆眨眼成赔钱货,又被人认出"养一院子怪东西、连黑市都栽过跟头的沈姑娘"而心虚,忽换嘴脸"晦气拿走拿走"几个铜板脱手逃走(误会式打脸)。带回家放破陶盆,一掉泪满院眼巴巴算账,沈宁却把珠随手扔进墙根装石子的破石臼"这是它的眼泪不是咱家的米"。夜里点破降服:"谁说要有用才能留下""你哭也好不哭也好都是我家的人,你的眼泪是你的谁也别想拿走",果冻头回毫无形象大哭一场、哭着哭着竟笑了(原来哭出来的可以不是钱、只是开心)。因软软凉凉晃晃悠悠像块果冻,得名**果冻**(没像雪糕梗脖反抗,只怯怯红眼问"这么好听的名字我配吗"——头回有个不因值钱、只因可爱才得来的名字),收为家中**老十**。庇护:被当『聚宝盆』逼哭采珠的孤独鲛人崽获救,头回安心睡在水里、不再怕哭不出珠就被丢掉。

## 7.5 配角与道具速查(陆续登场,随写随补)

- **古长老:** 万兽宗驭兽堂长老,灰袍山羊胡,怪脾气、眼高于顶,专管灵宠驯化。第3–4章因沈宁安抚暴走玄甲犀而赏识她,欲收其入驭兽堂被婉拒,反将拒绝脑补成「不慕虚名的高人风范」,赏三枚灵石。是「误会式打脸」的早期推手之一。
- **贺霆(第70–73章):** 内门富贵弟子,**萧战的兄弟**,锦袍玉带、从鼻孔看人、慕强护短。记恨沈宁害兄长下狱,于宗门品鉴会牵黑市强契暗影豹当众挑衅羞辱沈宁,反被沈宁咸鱼态度衬成小丑;为压她一头三次契印逼兽致暗影豹反噬暴走,被沈宁/糯糯当众降服(误会式打脸)。供出暗影豹购自"城里识货的人"(黑市线),与萧战同罪私通黑市,**下狱待查**。强契流又一块被打的招牌,亦坐实"强契流屡屡栽在沈宁手上"。
- **暗影豹(第71–73章):** 贺霆重金购自黑市斗兽场的三阶巅峰强契豹,通体漆黑、脖颈暗金契纹结痂。被贺霆契印反复灼烧逼战,品鉴会当众强契反噬暴走;沈宁听其"好痛别逼我"的求救,糯糯吞净其暴戾、沈宁言灵"谁问过你愿不愿意"抚平,当众对沈宁伏首落泪。**刻意未收入家**(避免与红豆=赤焰豹猛兽位重复、维持household十口):血脉崩裂暂交驭兽堂将养,去留日后由它自己定,沈宁留"养好想来随时管饭"的口子(温暖不入伙的庇护落点)。同时是又一桩坐实贺霆/萧战一脉私通黑市的活证据。
- **苏决:** 内门弟子,玄甲犀(三阶,强契)的主人。因强行催战导致契印反噬、爱宠暴走失控,当众失了脸面。`【后续可细化:与沈宁/萧战关系待定】`
- **老周(周伯):** 外门杂役,管这一片灯油柴米,腿脚不利索,是少数肯跟垫底的沈宁搭话的人。沈宁的日常信息源与「换灵米」对象,温情底色的配角。
- **玄甲犀事件(第3章):** 强契反噬的样板案例,坐实 §4.1「强契宠又强又恨、易反噬暴走」的设定,也是沈宁言灵能力首次公开使用、被误判为「驯兽天赋/深藏不露」的引爆点。
- **镇魂珠(黑市暗线初露,第5章钩子):** 黑市流出、封印某物的黑色小珠,表面有极细封纹,散发又冷又古老的气息。玄甲犀冲撞震松墙根后暴露在沈宁院墙角,被贪吃的糯糯一口吞下。**珠中所封 = 退化成黑奶猫的堕落妖王玄夜(见 §7)**,由此带出玄夜 arc 与黑市暗线。黑袍人丢失镇魂珠后惊惶,惧「封在里面的那位醒了」。
- **黑市探子·刀疤脸(第7章上门):** 黑袍人手下,刀疤横过半张脸的精瘦汉子,带两名同伙循气味摸到沈宁院墙塌口,凭残留凉气与新土判断珠子被「活物连泥带土一口吞了」。当面盘问沈宁,被糯糯肚中**玄夜首次外泄的威压(又冷又沉、裹着千年杀意的一眼)**当场吓退,误判沈宁为「能把镇魂珠当宝贝随便养在院里的深藏不露的高人」——又一桩误会式打脸,黑市暗线对沈宁的忌惮由此种下。后续(第9、11章)随黑市升级,刀疤脸一直作为「爷/钩爷」的下手跟班出场。
- **黑市买主「爷」(第9章夜探):** 黑袍人之上、钩爷之下的中层买主,腰悬黑铁令牌。带刀疤脸与一只强契掘地鼠「阿土」夜探沈宁院子,被沈宁言灵策反掘地鼠(问其「愿不愿意」令其抗命退缩)挫败,再误判沈宁深藏不露而撤,决定上报「请位真正的大人物」(即引出钩爷)。
- **黑市血脉炼师·钩爷(第11–12章上门):** 黑市资深血脉炼师,蜡黄面皮、包金牙、十指青灰(炼化痕迹),养一只被强契折磨到只剩食欲的血腥契宠「阿吼」。**不信「高人」之说,只信「味儿」**,循镇魂珠气息断定珠在糯糯腹中,提出剖团子取珠+赏十枚灵石。触发沈宁护短硬气(以命相挡)。玄夜破封后认出其为「妖王残魂」,惊惧弃珠而逃。逃当夜发密信入黑市深处:「封魂已醒,外门有变」——黑市暗线由此从「找珠」升级为「妖王苏醒」,引向更上层大佬(第三卷)。
- **玄夜苏醒进度(在糯糯腹中,第5–12章):** 镇魂珠入糯糯肚后,玄夜借糯糯打嗝半梦半醒(第5–7章);**第8章沈宁以言灵与玄夜首次正面对话**,得知团子肚里封着一位傲娇暴躁、暂时出不来的大爷;威压随糯糯不受控外泄护主(第7章吓退刀疤脸、第10章令萧战赤焰豹当众伏地)。**第11–12章封印破裂:糯糯被钩爷契宠所逼,「噗」一声把玄夜连同裂成两半的镇魂珠一起吐出,玄夜以巴掌大黑奶猫实体正式登场(0012,略早于原定 Arc2)。**因被封印+被糯糯「消化」大半,**残魂虚弱、道行尽失**,需在沈宁处慢慢养回(Arc 2 主线)。口头「只是暂居于此」,身体很诚实(冷、往沈宁怀里凑、被糯糯搂着睡)。
- **强契契宠求救伏笔(第4、10章):** 萧战的赤焰豹(三阶强契,怨恨主人)两度只对沈宁低声求救「带我走/放我走,他迟早把我送上斗兽场」,沈宁记下且区别于「团子乱吃」账——为后续策反赤焰豹/庇护线伏笔。
- **红豆(原赤焰豹,第22–25章策反归家):** 萧战的三阶强契宠,火红毛、幽红竖瞳、背上火纹。第22–23章契印被沈宁言灵激出旧恨而碎裂、反噬萧战,第24章俯首求随、被沈宁收为家中**排行老三**(老大糯糯、老二玄夜)。第25章正式安顿破院:斗兽场出身,只有编号「赤七」(第七笼红毛),早年的本名被磨忘;沈宁见其毛色「红得跟煮熟的红豆似的」,随口起名**红豆**(威风三阶妖兽却得软名,反差萌;呼应糯糯=汤圆)。性格:凶相是斗兽场逼出来的壳,内里护幼(给糯糯让食)、惜食(怕吃快被抢)、易满足。**是通往斗兽场暗线的活线索。**
- **斗兽场(第25章经红豆之口浮出全貌):** 黑市旗下,把灵宠当货,一笼一笼关着逼打,赢的接着打、输的拖去「后头那间屋子」炼血脉,进去没一个出来。屋里主炼者=**蜡黄脸、青灰十指的钩爷**(§7.5,其根扎在斗兽场)。斗兽场最里头一个笼子关着一块「会喘气的灰石头」,被当炼不出血的废物扔着等死,夜里自言自语「很老很老的、文绉绉的话」——**即退化古神咕噜(§7)的登场伏笔,引出 Arc 3。**
- **下月汰除考核(第4章起埋,逼近中):** 萧战约沈宁同场考核当众较量。第12章末点明考核将至,且黑奶猫玄夜将在考核日给众人「天大惊喜」(第38章玄夜再次叮嘱「考核那天带上本座,要给他们一个天大惊喜」)——第一卷收尾钩子,指向 Arc 4 萧战首轮打脸。
- **咕噜与钩爷的旧账(第34–37章):** 咕噜本名所属不详,黑市/斗兽场称钩爷为「钩三」(已无人再叫的旧名)。**钩爷十指青灰、炼瞎一只眼的来历由咕噜点破:** 是当年三炼咕噜不化、被咕噜这块「太古的根」反噬所致——青灰手是反噬刻进骨头的旧伤,一句「老夫记得你」即可勾起钻心剧痛。咕噜=上古古神残躯,虽退化成石、不能战,但「让你想起来」式的言语+地底封灵纹齐齐共鸣的余威,足以镇住钩爷。**镜头纪律:点到为止,不渲染炼化血腥。**
- **糯糯吞噬力首次实战展示(第35章):** 钩爷强契血腥契宠「阿吼」(被催战丹烧空、只剩食欲、言灵已喊不进)扑向糯糯,糯糯「啊呜」一口,把阿吼连同满身催战丹催出的暴戾气焰一并吞净;阿吼存活,从此「不饿了」、恢复神智。坐实糯糯=幼态饕餮「吞万物/吞攻击」之能,亦是面对「言灵失效之兽」的破局手段。钩爷因闻不透糯糯气味(太杂太老如深井)而首次失态后退。
- **斗兽场覆灭·清笼反转(第36–37章):** 钩爷喝令围攻,反被沈宁言灵逐头策反其契宠(「有人问它们想不想」),红豆破笼放出底下所有退化「废物」,斗兽场底当场崩塌。**「清笼」一语反转:清的是钩爷的笼。** 钩爷十指废、半身瘫坐,扬言上报黑市深处「真正的东家」(那位曾告诫手下「外门养镇魂珠的沈姑娘,能不沾就别沾」)。**反派收束:钩爷未死、斗兽场已废,黑市暗线向更上层大佬延续(第二卷)。**
- **收容所雏形(第38章):** 斗兽场底获释的退化灵宠无处可去,三三两两缀在沈宁身后跟回破院;沈宁一句「我家不大,破是破了点,管饭」松口接纳——**萌宠收容所正式起步,衔接第二卷主线。** 咕噜以「大馒头」之名加入家中,排行:糯糯(老大)、玄夜(老二)、红豆(老三)、咕噜随后。出斗兽场时晨光下「迎风迷眼」、被糯糯一句「大馒头哭啦」破功,为咕噜重见天日的情感落点。
- **汰除考核(第39–42章,第一卷高潮/Arc4):** 三年一回的逐人考核,五日后举行,末等者逐出宗门+没收灵宠(沈宁的「硬气开关」,只为护宠不得不赴考)。上回赵长老收银做局事发,这回主考换成**驭兽堂古长老**(灰袍山羊胡、眼高于顶,但坚信沈宁是「不慕虚名的高人」,全程把沈宁的随手之举脑补成高人风范——误会式打脸的放大器)。规矩:弟子携契宠上场,长老评定战力排名次。萧战当众下战书要与沈宁一较高下、算红豆旧账。
- **萧战的寒霜狼(第40–42章):** 萧战丢了赤焰豹后下血本买的新强契宠,雪白巨狼、冰蓝瞳、踏地结霜,三阶巅峰将破四阶。**同为黑市/斗兽场流出的强契宠**(脖子结痂契印),又强又恨主人,只求别被送回挨鞭的地方。考核当众被玄夜威压吓到伏地臣服,后经沈宁言灵点破其黑市来历,反成扳倒萧战的实证。考核后求随沈宁,收为家中**老四**(沿用红豆模式:跪求收留→沈宁叹气接纳→饭量危机的喜剧)。
- **玄夜「天大惊喜」兑现(第41章):** 考核前数日玄夜反常地猛吃猛睡、夜里对月攒劲。对决时当众放出**妖王本相**:浓夜色+黑焰漫开,一声「喵」即令三阶寒霜狼血脉臣服伏地。是第26章「撑两秒就力竭」之后**养回来的明显进阶**(契深/羁绊驱动的成长,非数值碾压),但仍只撑几息便力竭软倒,被沈宁不动声色揣回斗篷——满场误读为「高人随手放宠立威、立完即收」,沈宁深藏不露之名彻底坐实。口癖收尾:「本座,威风吧」。
- **萧战首轮系统性打脸(第42章,Arc4结算):** 言灵当众揭穿萧战两头强宠(赤焰豹、寒霜狼)皆出自黑市斗兽场,坐实其「与黑市勾连、戕害灵宠、败坏宗风」。古长老当场喝令看押萧战、禀掌门彻查其黑市首尾。**结果:萧战下狱待查;沈宁名次破天荒大涨,再无人敢叫她废物。** 打脸爽快、收束干净,不致郁。

### 表层线 —— 打脸
战兽榜垫底,面临逐出 / 没收灵宠 / 被迫上斗兽场;同门天才(如强契流的 **萧战**)屡次羞辱,后被打脸——他的强宠因怨恨在关键时刻被沈宁用言灵策反 / 自爆反噬。

### 情感暗线 —— 庇护(建议作为真正脊梁)
灵宠黑市、斗兽场、**血脉炼化**(杀宠夺力)。因为只有她听得见灵宠惨叫,她无法坐视,渐渐从"集邮"变成"灵宠庇护者",家里慢慢成了 **萌宠收容所**。
**沉重程度(已定):轻中度,不渲染血腥。** 黑市/斗兽场/血脉炼化是真实的暗线重量,给收容所存在的意义;但镜头落在「救出来之后的治愈」,施虐过程点到为止、靠暗示。反派打脸爽快、收束干净,**整体读感始终偏暖,不致郁**。

### 大谜团 —— 灵祖
言灵之力的来源 + "可爱=幼态神兽"的真相。背景设定:**千年前神兽集体陨落 / 封印,退化成凡间萌宠散落各地,只待言灵者唤醒。** 于是沈宁随手集邮,实为无意间重新集齐上古众神,牵动天下大局。

## 9. 叙事结构与节奏

- **单元式收宠:** 每个 arc = 一只 / 一群新宠 + 一个小地域 / 小反派 / 小故事(同源于作者既往作品的"外援"episodic 结构)。
- **底层穿插:** 一条黑市暗线 + 一条身世大谜团。
- **节奏曲线:**
  - 前期:治愈搞笑 + 小打脸(快速立人设、立招牌宠)
  - 中期:黑市与同门冲突升级(暗线浮出)
  - 后期:神兽觉醒 + 灵祖真相(大谜团收束)
