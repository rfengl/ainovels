# 第九十四章 管饱的席面

席面，终于还是吃上了。

驭兽堂后院，摆了满满几大桌。

沈宁找了个最角落、最清净的位置坐下，把她那一窝，一字排开。

这是她今天最有成就感的一刻。

什么战兽榜，什么破格上榜，都比不上眼前这几大盆冒着热气的吃食，来得实在。

“开动。”她言简意赅。

话音未落，糯糯已经一个猛子扎进了那盆灵米糕里。

“呜呜呜——好吃！太好吃了！”白团子吃得满脸都是，含混不清，“糯糯没骗人，糯糯真的能吃八个！”

暄暄一张嘴顶仨，呼噜呼噜，风卷残云。

锅巴这只全天下最挑食的鸟，难得没挑，对着一碟精致的灵果，吃得头也不抬，还不忘端着架子：“本凤……勉强赏脸。”

雪糕端着身段，矜持地，一小口一小口。

可那盆灵米羹见底的速度，出卖了它。

果冻泡在水里，捧着块小点心，吃一口，幸福地眯一下眼。

乐乐和愁愁共用一个胃，为了谁先吃，两个脑袋又掐了起来。

“我先！”
“不行，会撑死的，我们要撑死啦……”

酥酥怯生生地，把脸埋在尾巴里，只敢趁没人注意时，飞快地叼一口。

红豆和芝麻这两头威风的猛兽，蹲在桌边，吃相却斯文得很。

它们斗兽场出身，是真正饿过的。如今守着这一桌吃不完的饭，眼里是化不开的满足。

沈宁托着腮，看着这一幕，慢吞吞地，扒着自己碗里的饭。

值了。

为了这顿管饱的饭，挨那一架，值了。

可惜，这份清净，没能持续多久。

“沈姑娘。”

“沈师妹！”

“沈仙子在此用饭啊，可否赏脸……”

不知何时，那张最角落的桌子周围，竟围满了人。

全是宗门的弟子。

有内门的，有外门的，一个个堆着满脸的笑，捧着各色礼盒，往她跟前凑。

沈宁认得其中几张脸。

就在三个时辰以前，这几位还站在人群里，指着她那一窝，笑得最大声。

“一筐卖不出去的歪瓜裂枣”，就是其中一位说的。

如今，那位正一脸谄媚地，捧着一盒灵丹，要往她手里塞。

“沈师妹，一点心意，不成敬意。”

“您那只……那只白色的小团子，可真是灵秀啊。一看就知道是了不得的血脉。”

沈宁看了看怀里满嘴米糕渣的糯糯。

糯糯正茫然地，回望着那人，打了个饱嗝。

“……它叫糯糯。”沈宁说，“它就是个汤圆。”

“哎哟，您太谦虚了。”那人笑得更殷勤了，“深藏不露，深藏不露啊。”

沈宁:“……”

她放弃了解释。

这种事，她从前在斗兽场、在品鉴会上，已经领教过太多回了。

她说东，旁人偏要脑补成西。

她越说自家是歪瓜裂枣，旁人越觉得她高深莫测。

“沈师妹，我家中有几只灵宠，一直养不好，改日能否请您……”
“沈仙子，听闻您驯兽通神，不知可愿收徒……”
“沈姑娘，您那只小狐狸，我出三百灵石，您看……”

七嘴八舌的声音，嗡嗡地，糊了她一脸。

沈宁默默地，扒完了碗里最后一口饭。

然后，她站起身，拍了拍手。

“吃饱了。”她对自家那一窝说，“回家。”

她拎起水葫芦，捞起糯糯，招呼着剩下那一窝，头也不回地，从人群的缝隙里挤了出去。

留下一桌子捧着礼盒、面面相觑的弟子。

“……走了？”
“沈仙子这是……不为名利所动啊。”
“连三百灵石买狐狸都不动心，这份心境……”

身后的赞叹声，沈宁一个字都没听进去。

她只觉得，今天这饭，吃得是真累。

比打那一架还累。

——

回破院的路上，倒是清净了。

只是这清净里，透着点说不出的古怪。

一路上，遇见的弟子，再没人敢叫她“废柴”了。

他们远远地看见她，便自动让开一条道，眼神里满是敬畏，还有一种……打量着什么稀罕宝贝似的，火热。

沈宁被看得浑身不自在。

“咕噜。”她低声嘀咕，把怀里的青石头掂了掂，“他们看我干嘛。”

“看你呢。”咕噜慢悠悠地，叹了口气，“也看你这一窝呢。”

“老夫不是早说过么。你上了那个榜，便不再是没人理会的废柴了。”

“你这一院子东西，从今往后，”它浑浊的声音里，是化不开的凝重，“在旁人眼里，就是明晃晃的宝贝了。”

沈宁皱了皱眉。

她想起方才那个出三百灵石要买酥酥的人。

她下意识地，把怀里的水葫芦护紧了些。

“……宝贝就宝贝吧。”她哼了一声，“反正是我家的宝贝。谁也别想动。”

回到破院，那扇歪斜的院门，总算把外头那些火热的目光，关在了外面。

沈宁长舒一口气，瘫回了她那熟悉的门槛上。

总算到家了。

只是她没注意到，院门外不远处的墙角阴影里，今日不知何时起，多了一道陌生的、打量的目光。

那目光，盯着她这破院，盯了很久。

直到日头偏西，才悄无声息地，退了下去。

——

外门西头，那座院子里。

楚客卿听着属下的回报，脸色一点点地，沉了下去。

“锦城来人了？”

“是。”黑衣人低声道，“元家的二公子，元朗。打着‘观摩战兽榜、以宠会友’的旗号，今日午后，进的万兽宗。”

“可属下查过了，他在破院外头，盯了沈姑娘那院子，足足一个时辰。”

楚客卿端着茶盏的手，停住了。

元家。

锦城首屈一指的豪富，以收罗天下奇珍异宠为荣。那位元二公子，更是个出了名的、把灵宠当玩物的主儿。

他来得好快。

战兽榜的消息才传出去三日，外头的眼睛，就已经探进来了。

“果然。”楚客卿轻轻地，放下了茶盏，眼底是一片沉静的冷,“盯着她的，不止我一个。”

“千年前那位，也是这样。名声一起，四面八方的人，便都涌了过来。”

“有人想用她，有人想夺她的东西。”

他站起身，望向破院的方向。

“盯紧元朗。”他淡淡地吩咐，“看看这位元二公子，到底想从她那破院里，拿走点什么。”

“在我看明白之前……”

那双温和的眸子，又一次，沉淀下了那抹熟悉的锋芒。

“谁也别想，从她那儿，拿走任何东西。”
