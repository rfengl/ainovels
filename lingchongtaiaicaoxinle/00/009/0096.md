# 第九十六章 辨不出忠奸的废物

沈宁推开门，走了出来。

她没看元朗，径直走到那只华贵的兽笼前，蹲了下来。

笼子里那只青黑色的小东西，缩成一团，抖得更厉害了。

“别看我……”它把脑袋埋进短绒里，声音细弱，“求你别看我……我不是故意的……我也想辨得准……可我就是控制不住……”

沈宁听着它语无伦次的呜咽，眉头越皱越紧。

“它怎么了。”她抬起头，看向元朗。

元朗冷哼一声，余怒未消。

他本不想理会这个顶撞了他的丫头。

可转念一想，既然她对这废物有兴趣……

他眼珠一转，那点纨绔的算计，又活络了起来。

“沈姑娘也识货。”他皮笑肉不笑地，“不瞒你说，这小东西，是只獬豸。”

沈宁:“獬豸？”

墙根那块青石头，忽然“咯噔”动了一下。

咕噜慢悠悠地，挪到了门边，浑浊的目光，落在那笼子里的小东西身上。

“独角者，能触不直……”它压低了声音，喃喃自语，那语气里是说不出的复杂，“辨忠奸，明善恶，见人争斗便以角触理亏之人。好一脉法兽之裔。”

“怎么也落到了，这等人的笼子里。”

它的声音很轻，只有沈宁一个人，听得真切。

沈宁没作声，记下了。

她这一窝里的来历，每一只，都是咕噜这么一句一句，给她点破的。

“獬豸，神话里的法兽。”元朗那头，还在自顾自地说着，语气里满是嫌弃，“能辨忠奸，断善恶。我元家费了天大的力气，才寻来这么一只幼崽。”

“本想着养大了，做生意、断官司，谁是奸猾小人，它一眼便能瞧出来，那是何等的便利。”

他越说，脸越黑。

“可你猜怎么着。”他一脚踹在笼子上，那小东西吓得尖叫一声，“这是只废物！”

“辨忠奸？它辨个屁。”元朗咬牙切齿，“我带它见了多少正经客人、多少生意上的伙伴，它一个都辨不出来。”

“偏偏……”

他说到这儿，脸色古怪了一瞬，没再说下去。

沈宁却看明白了。

就在方才。

那只缩在笼子里、瑟瑟发抖的小獬豸，在元朗踹笼子的时候，曾极快地、极委屈地，抬起它那颗顶着小肉包的脑袋，朝着元朗的方向，轻轻地，顶了一下。

像是一种本能。

也像是一种，无声的控诉。

沈宁忽然就懂了。

她差点没忍住笑。

这哪是什么废物。

这只小獬豸，分明是只顶顶灵验的法兽。

它辨不出元朗那些“正经客人”和“生意伙伴”，是因为……那些人，本就没几个是好东西。

而它每一次，都执拗地，想去顶元朗。

因为在它那双辨忠奸的眼睛里，这满院子人中，心术最不正的，恰恰就是它的主人。

它不是辨不准。

它是辨得太准了。

准到，它的主人，怎么也不肯信。

“它没坏。”沈宁慢吞吞地开了口。

“嗯？”元朗一愣。

“我说，它没坏。”沈宁站起身，拍了拍手上的灰，“它好得很。”

“它辨不出你那些客人是好是坏，”她看着元朗，平静地说，“是因为它一直在告诉你一件事，你不肯听罢了。”

“什么事？”元朗皱眉。

沈宁顿了顿。

她本想说“它一直想顶的，是你”。

可她看了看元朗那张涨红的、不可一世的脸，又把话咽了回去。

懒得解释。

说了他也不信。

“没什么。”她摆摆手，“这只獬豸，多少钱，我买了。”

元朗:“……”

他先是一怔，随即，那阴沉的脸上，慢慢地，爬上了一抹算计的笑。

来了。

他就知道，这丫头方才不肯卖宠，是装清高。

如今，不也对他这“废物”动了心么。

“沈姑娘想要这只獬豸？”他慢条斯理地，把那只笼子，往身后挪了挪，“也不是不行。”

“它在我手里，本就是个累赘，正打算找个没人的地方，处置了。”

沈宁的眼神，冷了一下。

“不过……”元朗拖长了声音，享受着这难得的、扳回一城的快感，“白送，那是不可能的。”

“这样吧。”他抬起下巴，居高临下地看着她，“拿你那只小狐狸来换。”

“一只稀世的灵狐，换一只辨不出忠奸的废獬豸。沈姑娘，这买卖，你可是占了天大的便宜。”

他笃定，沈宁不会答应。

他要的，就是看她为了救这只小东西、又舍不得自家灵狐时，那副进退两难、低声下气来求他的窘样。

到那时，他丢的面子，自然就找回来了。

院门里头，缩在尾巴里的酥酥，听见“小狐狸”三个字，吓得浑身一抖，下意识地，又往后缩了缩，身形都开始变淡。

“对不起……”它小声地，带着哭腔，“是不是因为我……又要给宁宁添麻烦了……”

沈宁回头，看了酥酥一眼。

那只好不容易才不再说“对不起”的小狐狸，此刻又把脑袋埋了下去。

她的眼神，彻底沉了下来。

她转过身，看着元朗那张志在必得的脸。

“元公子。”她的声音，还是懒洋洋的，可那懒散底下，已经没了半分温度，“我再说最后一遍。”

“我家的人，一个,都不换。”

“至于这只獬豸……”

她瞥了一眼那只缩在笼子里、正怯怯望着她的小东西。

“你说要处置了它。”

“那我可就当真了。”
